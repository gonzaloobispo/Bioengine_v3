# Supervisión Remota: Antigravity Link para iOS

Según los documentos proporcionados, **Antigravity Link** es una extensión de código abierto (FOSS) diseñada para Google Antigravity que permite supervisar y orquestar agentes de IA de forma remota sin necesidad de estar frente al ordenador.

### 1. Instalación en el Entorno de Escritorio
Instala la extensión **Antigravity Link** directamente en tu instancia de escritorio de Google Antigravity. Esta extensión actúa como un servidor ligero que expone la interfaz de control de tus agentes.

### 2. Conexión desde iOS
*   **Red Local (Wi-Fi):** Conexión directa usando la dirección IP local de tu ordenador.
*   **Acceso Remoto:** Usa una VPN (como Tailscale) para acceder a tu máquina de forma remota y segura.

### 3. Funciones de Supervisión en iOS (Vibe Coding Móvil)
*   **Enviar Prompts y Comandos:** Corregir el rumbo o iniciar nuevas tareas desde el iPhone.
*   **Verificación Visual (Capturas):** Recibe capturas de pantalla en tiempo real del estado del IDE.
*   **Gestión de Modelos:** Cambia entre Gemini 3 Pro y otros modelos según la necesidad.
*   **Carga de Archivos:** Sube archivos desde tu móvil al entorno de trabajo del agente.

### Alternativa para Control Total: Sunshine + Moonlight
Para control granular (edición de líneas de código o uso de ratón en iPad), usa **Sunshine (Host) + Moonlight (Cliente iOS)** junto con Tailscale para streaming a 120 FPS con latencia casi nula.
